# Transparência e Consentimento Informado

## Acesso das diferentes partes envolvidas

- **Usuários autorizados do dashboard e API**: acesso claro aos dados disponibilizados, respeitando o nível de permissão atribuído (por exemplo, acesso a dados agregados ou precisos conforme perfil).
- **Administradores**: acesso restrito aos dados completos necessários para gestão do sistema, validação científica e manutenção da infraestrutura.
* **Parceiros institucionais**: acesso conforme acordos formais, garantindo uso ético e dentro das finalidades de conservação.

## Transparência na coleta de dados

- Mensagem clara nos treinamentos e documentações explicando quais dados são coletados (localização de fauna, eventos críticos, dados de sensores) e para qual finalidade.
- Disponibilização de documentação pública descrevendo a precisão, a frequência de atualização e as limitações dos dados.
- Indicação explícita de restrições, como a não divulgação de coordenadas exatas de espécies ameaçadas.

## Consentimento e comunicação com os usuários

- Possibilidade de solicitação de exclusão ou anonimização de determinados conjuntos de dados em casos específicos, conforme acordos firmados.
- Disponibilização de políticas de uso em formato claro e acessível, com linguagem técnica simplificada para facilitar o entendimento.
- Assinatura de Termos de Compromisso e Acordos de Uso de Dados por todos que tiverem acesso à API ou ao dashboard.

## Monitoramento contínuo

- Auditorias anuais internas e externas para verificar a conformidade com os acordos e políticas de uso de dados.
- Revisão periódica dos termos e políticas para alinhamento com melhores práticas de transparência e com novas regulamentações ambientais e de dados.

## Conclusão

Com práticas claras de comunicação, consentimento formalizado e auditorias regulares, o projeto garante que o uso das informações seja ético e transparente. Essa abordagem reforça a confiança no sistema, garante o uso ético das informações e contribui para a preservação segura e responsável da fauna.
