# ETL 
O processo de ETL (Extract, Transform, Load) é um fluxo utilizado em engenharia de dados para extrair, transformar e carregar informações em um banco de dados ou data warehouse. Ele ocorre em três etapas principais:

Extract (Extração): coleta de dados de diferentes fontes, como bancos de dados, APIs, planilhas, sensores ou arquivos de log.

Transform (Transformação): padronização, limpeza e enriquecimento dos dados, o que pode incluir remoção de duplicatas, tratamento de valores ausentes, conversão de formatos e cálculo de métricas.

Load (Carga): armazenamento dos dados transformados em um destino centralizado, como um banco de dados, data warehouse ou datalake, onde ficam prontos para análise.

Esse processo garante que informações dispersas e heterogêneas se tornem organizadas, consistentes e úteis para relatórios, análises avançadas e modelos de aprendizado de máquina.
